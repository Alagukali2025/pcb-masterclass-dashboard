New to PCB Design? Start Here.

A PCB (Printed Circuit Board) is like a sandwich of copper and insulating layers that connects electronic components. Think of each layer as a "floor" in a building — some floors carry electrical signals (signal layers), some supply power (power planes), and some act as shielding (ground planes). This guide walks you through every layer, why it matters, and the engineering rules that keep your board working reliably.

01 · METHODOLOGY
Single Source of Truth (SSOT)
One master stackup document that drives layout, simulation, and manufacturing — zero parameter drift between teams.
What is SSOT and why does it matter?

In a real engineering team, the layout engineer, the SI (Signal Integrity) simulation engineer, and the PCB fabrication vendor often use different dielectric constants or copper thicknesses — causing the manufactured board to have completely different impedance than what was simulated. An SSOT stackup file fixes this by being the single document that everyone reads from.

How to Build Your SSOT — Step-by-Step
1

Choose your layer count

Start with 4 layers for simple boards, 6–8 for mixed-signal, 10–12 for high-speed server boards. Always use an even number — this is required for stackup symmetry.

💡 Beginner tip: A 4-layer board (Signal / GND / Power / Signal) solves 90% of hobby and small commercial projects.
2

Select your base material (laminate)

For most designs: FR4. For signals above 3 GHz: Isola FR408HR or Megtron 6. For mmWave RF: Rogers RO4350B. Your material choice locks in the dielectric constant (Dk).

💡 Dk is like the "slowness factor" for electricity — higher Dk = slower signal propagation and more compressed impedance.
3

Define copper weights per layer

Outer layers: ½ oz (0.7 mil) for fine traces. Inner signal layers: ½ oz. Plane layers (GND/PWR): 1 oz (1.4 mil) for current capacity. Write the weight for every layer in your SSOT.

4

Set dielectric thicknesses to hit target impedance

Use an impedance calculator (like the one below) to pick dielectric heights (H) and trace widths (W) that produce 50Ω single-ended or 100Ω differential. Document the exact trace width and H for each layer.

💡 Thinner dielectric = narrower trace for same impedance. This is how you get fine-pitch routing on high-density boards.
5

Verify symmetry and lock the file

Mirror your prepreg and core selections about the board midplane — identical glass style and thickness above and below. Then version-control the file (Git, PLM, or even a dated PDF). Any change requires a new revision with sign-off.

02 · STRUCTURE
Anatomy of a PCB Stackup
Hover over each layer below to understand what it does. Every layer in a PCB has a specific job.
8-LAYER PROFESSIONAL STACKUP — IPC-4101C · 62 MIL NOMINAL
TOP SILK Silkscreen — Component Labels & Markings Ink, ~0.1 mil
White or yellow ink printed on top of soldermask. Contains reference designators (R1, C4, U2), component outlines, and board labels. Has zero electrical function — purely for assembly and debug.
TOP SM Soldermask — LPI Green Lacquer 0.5–1.2 mil
Liquid Photo-Imageable (LPI) coating that protects copper traces from oxidation and solder bridges. The green color of most PCBs comes from this layer. Openings (apertures) expose pads for soldering components.
L1 — SIGNAL ½ oz Copper — Top Signal Layer (Microstrip) 0.7 mil / 17.5 µm
Outer signal layer. Traces here are microstrip geometry — copper on top of dielectric, with air above. Component pads, short traces, RF routing. ½ oz copper allows 3 mil minimum trace width for fine-pitch ICs.
DIELECTRIC Prepreg 2116 — Dk 3.9, Df 0.020 (FR4) 4.0 mil
Pre-impregnated glass fiber + resin. Bonds L1 copper to the L2 core during lamination under heat and pressure. The "2116" refers to the glass weave style — medium density, good for impedance control. Dk 3.9 at 1 GHz.
L2 — GND 1 oz Copper — Solid Ground Reference Plane 1.4 mil / 35 µm
Primary ground reference plane. Must be SOLID — no gaps, no slots. This layer provides the return current path for every signal on L1 and L3. It also defines the impedance of L1 microstrip traces by setting dielectric height H.
CORE FR4 Core — Dk 4.2, Tg 170°C 10.0 mil
Rigid glass-epoxy core. The structural backbone of the stackup. "Core" is pre-cured (hard) laminate, unlike prepreg (soft until laminated). Dk 4.2 at 1 GHz. Higher Tg (glass transition temp) means more heat resistance during reflow.
L3 — SIGNAL ½ oz Copper — Inner Signal Layer (Stripline) 0.7 mil / 17.5 µm
Buried inner signal layer. Traces here are stripline geometry — fully enclosed in dielectric above and below. Better EMI shielding than microstrip. Use for high-speed differential pairs: PCIe, DDR, Ethernet. 100% surrounded by Dk material.
DIELECTRIC Prepreg 1080 — Dk 3.8, Df 0.018 (FR4) 3.0 mil
Thinner "1080" glass style prepreg. Lower resin content than 2116. Used for tighter dielectric control. 3 mil height allows narrower trace widths for a given impedance target, enabling denser routing channels.
L4 — PWR 1 oz Copper — Power Distribution Plane 1.4 mil / 35 µm
Power plane. Carries VCC, 1V8, 3V3, or other rails as solid copper pours. The tight spacing between L4 (power) and L5 (GND) creates a distributed capacitance that helps suppress power distribution noise (PDN decoupling).
← BOARD MIDPLANE — SYMMETRY AXIS →
L5 — GND 1 oz Copper — Secondary Ground Plane 1.4 mil / 35 µm
Secondary ground plane. Mirror of L4. Forms a tight GND-PWR pair with L4 for power integrity. Also acts as the lower reference plane for L6 stripline signals. Symmetric with L4 copper weight about the midplane.
DIELECTRIC Prepreg 1080 — Dk 3.8 (Mirror of above) 3.0 mil
SYMMETRIC MIRROR: Must be identical glass style (1080) and thickness (3.0 mil) as the prepreg above midplane. Asymmetry here is the #1 cause of board warpage during reflow soldering.
L6 — SIGNAL ½ oz Copper — Inner Signal (Best Shielded) 0.7 mil / 17.5 µm
Best-shielded inner signal layer — flanked by GND planes on both sides. Route your most sensitive high-speed signals here: DDR5 DQS strobe, PCIe Gen 4+ lanes, 10GbE. Maximum isolation from external EMI sources.
CORE FR4 Core — Dk 4.2, Tg 170°C (Mirror) 10.0 mil
SYMMETRIC MIRROR: Must be identical to the core above midplane (same material, same thickness). This is the other half of the structural sandwich. Equal CTE (Coefficient of Thermal Expansion) prevents differential stress during reflow.
L7 — GND 1 oz Copper — Ground Reference Plane 1.4 mil / 35 µm
Reference plane for L8 (bottom microstrip). Mirror of L2. Provides return current path for bottom-side component signals. Connect to L2, L5 via stitching vias every λ/20 to suppress resonant cavity modes.
DIELECTRIC Prepreg 2116 — Dk 3.9 (Mirror of top) 4.0 mil
SYMMETRIC MIRROR: Identical to top 2116 prepreg. Same thickness (4.0 mil) and same glass style. This ensures the board expansion behavior is identical top and bottom.
L8 — SIGNAL ½ oz Copper — Bottom Signal Layer (Microstrip) 0.7 mil / 17.5 µm
Bottom outer signal layer. Microstrip geometry, mirror of L1. Typically used for power connector escapes, low-speed I/O, LED drive signals. Bottom-side SMT components solder to pads here.
BOT SM Soldermask — LPI Green Lacquer (Bottom) 0.5–1.2 mil
Bottom soldermask. Same function as top — protects copper, prevents solder bridges. Some designs use different colors (red, blue) on bottom for visual identification during assembly.

03 · MATERIALS
Material Selection — SI & PI Impact
Your choice of laminate determines signal loss, operating frequency limit, and cost. Choose wrong and no amount of clever routing can fix it.
📡 Beginner Concept: Dk and Df. Dk (Dielectric Constant) controls how fast signals travel. Df (Dissipation Factor / loss tangent) controls how much signal energy is absorbed by the material as heat. Lower Df = cleaner signals at high frequencies. FR4's Df is ~20× worse than Rogers at 10 GHz.

🏡

What is Dk?

Dielectric Constant — how much the material slows down an electromagnetic wave compared to vacuum. Dk=4 means the signal travels at 50% the speed of light inside the material.

Like running through water (Dk=4) vs. running on a track (vacuum, Dk=1).
🔥

What is Df?

Dissipation Factor (loss tangent) — how much signal energy the dielectric absorbs and converts to heat. At 10 GHz, FR4 (Df=0.02) absorbs 5× more than Megtron 6 (Df=0.004).

Like speaking through cardboard (FR4) vs. through open air (Rogers).
🌡️

What is Tg?

Glass Transition Temperature — the point where the material softens from rigid to rubber. Lead-free reflow peaks at 260°C, so you need Tg > 150°C minimum. High-Tg FR4 (170°C) is the standard choice.

Like chocolate — solid below its "Tg", soft and deformable above it.
04 · ELECTRICAL
Impedance Control
Characteristic impedance is the most important electrical property you design into a PCB trace. Mismatch causes signal reflections, data errors, and EMI.
⚡ Beginner Concept: Why does 50Ω matter? When you plug headphones into a phone, impedance mismatch causes distortion. For high-speed digital signals (USB, PCIe, HDMI), impedance mismatch causes reflections — the signal bounces back along the trace and corrupts data. Controlling impedance to ±10% is how you prevent this.
Microstrip — Surface Trace

Trace is on the outer surface. Electric field is partly in air (above) and partly in dielectric (below). This makes εr_eff < εr. Faster propagation, easier to route, but more EMI exposure.

Stripline — Buried Trace

Trace is completely inside the dielectric. εr_eff = εr (full material Dk). Better shielding, lower crosstalk, lower EMI emission. Slower propagation than microstrip. Best for critical signals.

Impedance Formulas — Wadell / IPC-2141A
① MICROSTRIP (Surface Trace)
W_eff = W + (T/π) × ln(4e / √((T/H)² + (T/πW)²))
Z₀ = (87 / √(εr + 1.41)) × ln(5.98×H / (0.8×W_eff + T))
② STRIPLINE (Dual Ref Plane)
Z₀ = (60 / √εr) × ln(4×b / (0.67×π×(0.8×W + T)))
where b = B1 + B2 (total dielectric between reference planes)
③ DIFFERENTIAL MICROSTRIP (Edge-Coupled Pair)
Zodd = Z₀_single × (1 − 0.347 × e^(−2.9×S / (2×H)))
Zdiff = 2 × Zodd

05 · MECHANICAL
Stackup Symmetry & Warpage
An asymmetric stackup will warp during reflow like a bimetallic strip. IPC-6012 permits max 0.75% bow/twist for standard boards.
⚠️ Beginner Warning: The #1 warpage cause in production is engineers who specify symmetric thicknesses but use different glass styles (e.g., 2116 above midplane, 7628 below). Glass style determines the CTE — even if total thickness matches, different glass styles expand differently during reflow. Always mirror the glass style number, not just the thickness.

06 · SI & EMC
Reference Planes
A solid, uninterrupted ground plane is the single most impactful SI and EMC decision in your entire stackup design.
🧲

Why Return Currents Matter — The Key Concept

Every signal current needs a return path. At DC, this return current spreads across the entire ground plane. But at high frequencies (>1 MHz), the return current concentrates directly beneath the signal trace — following the path of least inductance, not least resistance.

If you cut a slot in the ground plane under a trace, the return current must detour around the slot — creating a large current loop, which radiates like an antenna and causes EMI failures. This is called "return current discontinuity" and it's responsible for the majority of first-article EMC test failures.
8-Layer Layer Assignment — Best Practice
L1 — Signal (Microstrip) · Short escapes, ICs, RF · Reference: L2 GND
L2 — SOLID GND Reference · Primary return plane · MUST be uninterrupted
L3 — Signal (Stripline) · PCIe, DDR DQ, USB · Refs: L2 GND + L4 PWR
L4 — Power Plane · 1V0 / 1V8 / 3V3 rails · Tight to L5 for PDN capacitance
L5 — SOLID GND Reference · Secondary GND + PDN pair with L4
L6 — Signal (Stripline) · BEST SHIELDED · DDR5 DQS, PCIe Gen4+ · Refs: L5+L7 GND
L7 — SOLID GND Reference · Mirror of L2
L8 — Signal (Microstrip) · Power connectors, low-speed I/O · Reference: L7 GND

07 · INTERCONNECT
HDI & Via Types
Vias are how signals travel between layers. Choosing the wrong via type is one of the most expensive mistakes in PCB design — via changes require re-lamination.

Through-Hole Via

Drills all layers top-to-bottom. Most common, lowest cost. Max aspect ratio 10:1. Stubs cause reflections above 6 Gbps — back-drill to remove.

Blind Via

Connects an outer layer to one or more inner layers without going all the way through. Laser-drilled for microvias. Required for 0.5mm pitch BGA escape.

Buried Via

Inner layers only — not visible from outer surface. Requires sequential lamination. High cost. Frees surface routing channels for densest BGA designs.

Microvia (HDI)

Laser-drilled, ≤6 mil diameter. Aspect ratio must be ≤0.75:1 (IPC-2226). Stackable (filled with copper). Required for 0.4mm BGA on any-layer HDI.

08 · STANDARDS
IPC Standards Reference
IPC standards are the contractual language between designers, fabricators, and assemblers. Cite these in your SSOT document for zero-ambiguity manufacturing.
IPC-2221B

Generic Standard on PCB Design

Foundation standard for all PCB designs. Covers minimum conductor spacing, annular ring requirements, via aspect ratios, thermal relief pads, and electrical clearances for all voltage levels. Every PCB must comply with this document.

IPC-4101C

Base Materials for Rigid PCBs

Defines material grade "slash-sheet" designators: /21 = standard FR4, /24 = high-Tg FR4, /99 = halogen-free. Always specify laminate by IPC-4101 designator in your SSOT — never by brand name alone. Prevents unauthorized substitutions at the fab house.

IPC-4562

Metal Foil for Printed Wiring Board

Defines copper foil grades: ED (electrodeposited — standard), RA (rolled annealed — flexible circuits), VLP (Very Low Profile — mandatory for ≥10 GHz to minimize surface roughness conductor loss). Specify VLP copper in your SSOT for high-speed designs.

IPC-2226

Sectional Design Standard for HDI

Covers microvia design rules, build-up layer specifications, land pad sizing for stacked vs. staggered microvias, and capture pad annular ring requirements. Essential for any design with BGA pitch below 0.65mm.

IPC-6012E

Qualification & Performance for Rigid PCBs

Defines Class 1/2/3 acceptance criteria for bow/twist (<0.75% Class 2, <0.5% Class 3), annular ring minimum, hole wall quality, dielectric thickness tolerance, and copper plating thickness. Specify Class 3 for aerospace/medical, Class 2 for standard commercial.

IPC-1601

Printed Board Handling and Storage

Mandates moisture bake-out (125°C for 4–8 hours minimum) before reflow soldering. Non-compliance is the primary cause of measling, delamination, and CAF (Conductive Anodic Filament) failures. Must be called out explicitly on your fab drawing.

// PCB Impedance Calculator — Wadell / IPC-2141A // All dimensions in mils. Output in Ohms. function calcMicrostrip(er, W, T, H) { const Weff = W + (T / Math.PI) * Math.log(4 * Math.E / Math.sqrt( Math.pow(T/H, 2) + Math.pow(T/(Math.PI*W), 2) )); return (87 / Math.sqrt(er + 1.41)) * Math.log(5.98 * H / (0.8 * Weff + T)); } function calcStripline(er, W, T, B1, B2) { const b = B1 + B2; // total dielectric between GND planes return (60 / Math.sqrt(er)) * Math.log(4*b / (0.67*Math.PI*(0.8*W + T))); } function calcDiffMicrostrip(er, W, T, H, S) { const Z0 = calcMicrostrip(er, W, T, H); const Zodd = Z0 * (1 - 0.347 * Math.exp(-2.9*S/(2*H))); return { Z0, Zodd, Zdiff: 2 * Zodd }; } // Example: 50Ω microstrip on FR4 const Z = calcMicrostrip(4.1, 5.0, 1.4, 4.0); console.log(`Z0 = ${Z.toFixed(1)} Ω`); // → ~50 Ω // Example: 100Ω diff pair on FR4 const d = calcDiffMicrostrip(4.1, 4.5, 1.4, 4.0, 5.0); console.log(`Zdiff = ${d.Zdiff.toFixed(1)} Ω`); // → ~100 Ω // Tolerance check — IPC-2141A ±10% function checkTolerance(Z0, target, tolerancePct = 10) { const error = Math.abs(Z0 - target) / target * 100; return { pass: error <= tolerancePct, errorPct: error.toFixed(1), range: [target * (1-tolerancePct/100), target * (1+tolerancePct/100)] }; }
